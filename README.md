# Weather Time

## Purpose
Provide travelers with a convienient source to discover the current and future forecast of any city. This application can be used for local and international weather. Destination is never a problem.

## Built With
* HTML
* CSS
* JavaScript
* Bootstrap
* 3rd Party API Calls
